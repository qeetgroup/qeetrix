# Qeetix

**Qeetix** is the shared design system for **Qeet Group** products. It is built on
**Tailwind CSS v4** and **shadcn/ui with the Base UI registry**, and themed by a single,
token-driven source of truth.

## Packages

- **`@qeetix/tokens`** — design tokens (W3C JSON → Style Dictionary → CSS + JSON). Generates the
  `:root` / `.dark` runtime variables and a `--qx-`-prefixed raw export.
- **`@qeetix/ui`** — the React component library (this workshop).
- **`@qeetix/brand`** — the Qeet logo (theme-adaptive) and custom icons.

## Using it

```tsx
import { ThemeProvider, Button } from "@qeetix/ui";
import "@qeetix/ui/styles.css";

<ThemeProvider defaultTheme="system">
  <Button>Authenticate with Qeet</Button>
</ThemeProvider>;
```

## Theming

Light and dark are first-class and driven by the `.dark` class on the root element — flip the
**Theme** toolbar above to preview either. Colours are currently a neutral/greyscale ramp; the
brand palette is a documented open decision (OD-DS-03) and drops in by editing only the primitive
ramps in `@qeetix/tokens`.

See **Foundations → Colors / Typography / Spacing & Radius** for the token galleries, and
**Primitives** / **Brand** for components.
